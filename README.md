# v1-basic-calculator.js

This project is a **basic JavaScript calculator** built using only core JavaScript concepts.
It is part of my learning journey while practicing conditions, user input, and basic logic.

The calculator works entirely with `prompt()` and `alert()` and does not use HTML buttons or the DOM.

## ğŸ§® How It Works

1. The user enters the first number
2. The user chooses an operator (`+`, `-`, `*`, `/`, `**`, `%`)
3. The user enters the second number
4. The program:
   - Checks if the inputs are valid numbers
   - Prevents division or modulo by zero
   - Executes the selected operation
5. The result is shown using an alert
